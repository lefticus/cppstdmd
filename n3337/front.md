This document consists of the working draft for the ISO standard for the
programming language C++ with all the changes made through the October,
2006 meeting of WG21, as reflected in WG21 document number N2134, with
the addition of placeholders for the following new features:

- Concurrency memory model, Clause  [[intro.concur]]
- Thread-local storage, Clause  [[basic.stc.threadlocal]]
- Programmer-controlled garbage collection, Clause 
  [[basic.stc.collect]]
- Concepts, Clause  [[temp.concepts]]
- Atomic operations library, Clause  [[atomics]]
- Thread support library, Clause  [[thread]]
