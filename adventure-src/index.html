Welcome

Welcome to the C++ Standard Adventure!

Use the terminal on the left to explore the C++ standard documentation as an interactive world.

Quick Start

  • look - View your current location
  • go north/south/east/west - Navigate between sections
  • enter <area> - Enter a sub-section
  • exit - Return to parent section
  • timeshift cpp20 - Travel to a different C++ era
  • help - See all commands

About

This game transforms the C++ standard into an explorable world. Each section becomes a location you can visit. Travel through time to see how C++ has evolved from C++11 to C++26.
